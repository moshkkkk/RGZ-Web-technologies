# 🎬 VideoPlatform

**Видеоплатформа для потокового вещания**

## 📖 О проекте

**VideoPlatform** — это современная веб-платформа для просмотра, загрузки и управления видеоконтентом.

Проект выполнен в рамках расчётно-графической работы по дисциплине **«Информационные технологии»**.

---

## ✨ Основные возможности

| Функция | Описание |
|---------|----------|
| 🔐 Аутентификация | Регистрация и вход с использованием JWT-токенов |
| 📤 Загрузка видео | Поддержка форматов MP4, MOV, AVI |
| 🎥 Потоковое видео | Воспроизведение с возможностью перемотки |
| 💬 Комментарии | Обсуждение видео под каждым роликом |
| 👁️ Просмотры | Автоматический счётчик просмотров |
| 🎨 Современный дизайн | Звёздное небо-стиль с анимациями |

---

## 🛠️ Технологии

**Backend:**
- Python 3.10
- Django 4.2
- Django REST Framework
- JWT (JSON Web Tokens)

**Frontend:**
- React 18
- Axios
- CSS3 

**База данных:**
- PostgreSQL / SQLite
